position static

position static digunakan untuk mengatur element menjadi static secara default. Elemen akan mengikuti posisi normal secara default, elemen tidak dipengaruhi oleh properti seperti top, bottom, left dan right.

contoh :

Elemen ini diatur menggunakan posisi static

position relative

Sebuah element HTML yang menggunakan posisi relative akan terletak pada posisi normal. Mengubah nilai properti top, right, bottom, dan left dari elemen yang diposisikan "relative" akan membuatnya jauh dari posisi normalnya.

contoh :

Elemen ini diatur menggunakan posisi : relative;

position fixed

Sebuah element HTML yang di setting dengan position fixed akan memiliki sifat tetap, tanpa ada perubahan bahkan jika halaman website di scroll. Properti left, bottom, top dan right dapat digunakan pada position fixed

contoh :

Elemen ini diatur menggunakan posisi : fixed















position absolute

Element HTML yang menggunakan position absolute akan diposisikan relatif dengan element lain yang mendahului nya yang terdekat. Bukan relatif terhadap layar secara normal.

contoh :

Elemen ini diatur menggunakan position: relative;
Elemen ini diatur menggunakan posisi: absolute













position sticky

Contoh ini untuk sticky, untuk mengetahui bagaimana posisi sticky bekerja.

cobalah untuk scroll untuk melihat sticky menempel diatas.

Saya adalah sticky!

Dalam contoh ini, elemen sticky menempel pada posisi paling atas halaman (top:0).

Cobalah untuk melakukan scroll...

Catatan: IE/Edge 15 dan versi lebih lama tidak mendukung sticky.
